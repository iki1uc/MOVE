# MOVE · ANKER · Stabilisationsachse · Eigenständige NC-Engine

MOVE ist eine eigenständige NC-Engine, die nicht mehr auf geliehene Module
angewiesen ist. Dieses Repository enthält:

- eine eigene MOVE.core.js Engine
- eine eigene ANKER-Stabilisationsachse
- eine eigene ID-Geometrie (id.html)
- eine eigene Bewegungs-Geometrie (index.html)
- eigene tmp-Achsen für Lage, Ort, Home, Delta, Stabil

MOVE ist jetzt vollständig unabhängig und kann als Basis für
OCTA, WHIRL, REAL, Quadrant und RESPO dienen.
